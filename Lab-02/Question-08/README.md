# Question 8 – Count Safe Current Readings

## Problem Statement

Write a C++ program that accepts 8 current readings and counts how many are safe and unsafe.

## C++ Concepts Practised

- for loops
- if...else statements
- Counters
- User input and output

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter 8 current readings.
4. View the number of safe and unsafe readings.

## Sample Output

Readings = 5 6 2 14 5 4 3 5.6

Safe Readings = 7

Unsafe Readings = 1

## What I Learned

I learned how to combine loops and conditional statements to count occurrences based on a condition.
