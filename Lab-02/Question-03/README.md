# Question 3 – Scholarship Eligibility for Engineering Student

## Problem Statement

Write a C++ program that determines whether a student qualifies for an engineering scholarship based on GPA.

## C++ Concepts Practised

- if...else statements
- User input and output
- Decision making
- Floating-point variables

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter the GPA.
4. View the scholarship eligibility result.

## Sample Output

Enter GPA: 3.8

Eligible for engineering scholarship

## What I Learned

I learned how to use conditional statements to determine eligibility based on a given requirement.
