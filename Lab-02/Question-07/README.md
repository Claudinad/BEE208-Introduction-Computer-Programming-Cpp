# Question 7 – Average Voltage from Sensor Readings

## Problem Statement

Write a C++ program that accepts 10 voltage readings from a sensor and calculates the average voltage.

## C++ Concepts Practised

- for loops
- User input and output
- Arithmetic operations
- Accumulators

## Formula Used

Average Voltage = Total Voltage ÷ Number of Readings

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter the 10 voltage readings.
4. View the average voltage.

## Sample Output

Enter Voltage reading = 2 3 4 5 6 7 8 9 1 13

Average Voltage = 5.8 V

## What I Learned

I learned how to use a for loop and accumulator variable to calculate the average of multiple values.
