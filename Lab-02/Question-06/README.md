# Question 6 – Series Resistor Calculator

## Problem Statement

Write a C++ program that accepts five resistor values connected in series and calculates the total resistance.

## C++ Concepts Practised

- for loops
- Accumulators
- User input and output
- Arithmetic operations

## Formula Used

Total Resistance = R1 + R2 + R3 + R4 + R5

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter the five resistor values.
4. View the total resistance.

## Sample Output

Enter resistor values: 10 20 30 40 50

Total Resistance = 150 Ohms

## What I Learned

I learned how to use a for loop to repeatedly collect inputs and calculate a running total.
