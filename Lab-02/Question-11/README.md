# Question 11 – Password Access to Control Panel

## Problem Statement

Write a C++ program that repeatedly asks the user to enter a password before granting access to an electrical control panel. The correct password is 2080.

## C++ Concepts Practised

- while loops
- User input and output
- Conditional statements
- Loop control

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter the password.
4. Continue entering passwords until the correct password (2080) is entered.

## Sample Output

Enter password: 1234

Incorrect password. Try again.

Enter password: 2080

Access granted to control panel.

## What I Learned

I learned how to use a while loop to repeatedly request input until a correct value is entered and how to implement simple password verification in C++.
