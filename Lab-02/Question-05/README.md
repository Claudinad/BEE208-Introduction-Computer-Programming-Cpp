# Question 5 – Power Rating Menu

## Problem Statement

Write a C++ program that performs electrical calculations selected from a menu.

## C++ Concepts Practised

- switch statements
- Arithmetic operations
- User input and output
- Menu-driven programming

## Formula Used

Power = Voltage × Current

Resistance = Voltage ÷ Current

Energy = Power × Time

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Select a calculation.
4. Enter the required values.
5. View the result.

## Sample Output

Enter choice: 1

Enter voltage: 12

Enter current: 2

Power = 24 W

## What I Learned

I learned how to use switch statements to perform different calculations based on user choices.
