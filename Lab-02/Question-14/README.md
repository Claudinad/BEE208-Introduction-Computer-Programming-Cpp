# Question 14 – Skip Invalid Temperature Readings Using Continue

## Problem Statement

Write a C++ program that accepts 10 temperature readings from an electronic device. Temperature readings below 0°C are considered invalid and should not be included in the average calculation.

## C++ Concepts Practised

- for loops
- continue statement
- Conditional statements
- User input and output
- Average calculation

## Condition Used

Invalid temperature condition:

Temperature < 0°C

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter the 10 temperature readings.
4. Invalid readings will be skipped automatically.
5. View the average of valid temperature readings.

## Sample Output

Enter temperature reading 2: -5

Invalid reading.

Enter temperature reading 5: 48

I learned how to use the continue statement to skip unwanted values during loop execution and calculate results using only valid data.
