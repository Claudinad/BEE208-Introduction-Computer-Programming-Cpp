# Question 2 – Current Overload Checker

## Problem Statement

Write a C++ program that compares measured current with rated current and determines whether an overload condition exists.

## C++ Concepts Practised

- if...else statements
- User input and output
- Comparison operators
- Decision making

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter the rated current.
4. Enter the measured current.
5. View the result.

## Sample Output

Enter rated current: 10

Enter measured current: 12

Overload detected. Circuit breaker should trip.

## What I Learned

I learned how to compare two values and make decisions using if...else statements.
