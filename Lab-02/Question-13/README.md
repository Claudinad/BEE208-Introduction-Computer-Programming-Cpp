# Question 13 – Stop Test on Low Battery Using Break

## Problem Statement

Write a C++ program that accepts up to 10 battery voltage readings and stops immediately when a low battery is detected.

## C++ Concepts Practised

- for loops
- break statement
- Conditional statements
- User input and output
- Loop control

## Condition Used

Low battery condition:

Voltage < 10.5 V

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter battery voltage readings.
4. The program stops automatically when a low battery is detected.

## Sample Output

Enter battery voltage reading 1: 12
Battery voltage normal.

Enter battery voltage reading 2: 11
Battery voltage normal.

Enter battery voltage reading 3: 10.2
Low battery detected. Test stopped.

## What I Learned

I learned how to use the break statement to terminate a loop immediately when a specific condition occurs.
