# Question 4 – Electrical Component Menu

## Problem Statement

Write a C++ program that displays a menu of electrical components and shows a description based on the user's choice.

## C++ Concepts Practised

- switch statements
- User input and output
- Menu-driven programs
- Decision making

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Select a menu option.
4. View the component description.

## Sample Output

Enter choice: 1

Resistor: Used to limit current in a circuit

## What I Learned

I learned how to create menu-driven programs using switch statements.
