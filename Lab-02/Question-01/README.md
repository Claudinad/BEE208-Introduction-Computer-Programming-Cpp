# Question 1 – Voltage Level Classifier

## Problem Statement

Write a C++ program that accepts a voltage reading and classifies it as Low, Normal, High, Dangerous, or Invalid.

## C++ Concepts Practised

- if...else if...else statements
- User input and output
- Relational operators
- Decision making

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a voltage value.
4. View the voltage classification.

## Sample Output

Enter voltage: 240

Normal voltage

## What I Learned

I learned how to use multiple conditions with if...else if...else statements to classify values into different categories.
