# Question 10 – Battery Voltage Monitoring

## Problem Statement

Write a C++ program that repeatedly accepts battery voltage readings and stops when the voltage drops below 12 V.

## C++ Concepts Practised

- while loops
- User input and output
- Conditional statements
- Loop control

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter battery voltage readings.
4. Continue until a reading below 12 V is entered.

## Sample Output

Battery Voltage = 13 V

Battery Voltage = 12.5 V

Battery Voltage = 11 V

Battery voltage low. Recharge required.

## What I Learned

I learned how to use a while loop to continuously process data until a stopping condition is reached.
