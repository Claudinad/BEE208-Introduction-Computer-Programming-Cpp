# Question 15 – Electrical Load Limit Checker

## Problem Statement

Write a C++ program that continuously accepts appliance power ratings and adds them to the total load until the load exceeds 3000 W.

## C++ Concepts Practised

- while loops
- User input and output
- Accumulator variables
- Conditional statements
- Loop control

## Formula Used

Total Load = Sum of Appliance Power Ratings

## Condition Used

Load Limit:

Total Load > 3000 W

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter appliance power ratings one after another.
4. Continue until the total load exceeds 3000 W.
5. View the warning message and final load.

## Sample Output

Enter appliance power rating: 1000

Current Total Load = 1000 W

Enter appliance power rating: 800

Current Total Load = 1800 W

Enter appliance power rating: 1500

Current Total Load = 3300 W

Load limit exceeded. Do not add more appliances.

Final Total Load = 3300 W

## What I Learned

I learned how to use a while loop and an accumulator variable to continuously process input until a specified limit is exceeded.
