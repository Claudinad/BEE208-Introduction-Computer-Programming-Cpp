# Question 9 – Current Monitoring System

## Problem Statement

Write a C++ program that continuously accepts current readings until an overcurrent condition is detected.

## C++ Concepts Practised

- while loops
- User input and output
- Conditional statements
- Repetition structures

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter current readings.
4. Continue entering values until a reading above 10 A is entered.

## Sample Output

Enter Current Reading = 5

Safe Current Reading = 5 A

Enter Current Reading = 8

Safe Current Reading = 8 A

Enter Current Reading = 9

Safe Current Reading = 9 A

Enter Current Reading = 11

Overcurrent detected. Monitoring stopped.

## What I Learned

I learned how to use a while loop to repeatedly process input until a specific condition becomes false.
