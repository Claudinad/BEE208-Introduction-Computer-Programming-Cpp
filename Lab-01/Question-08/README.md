# Question 8

## Problem Statement

Write a C++ program that reads an integer from the user and determines whether it is odd or even.

## C++ Concepts Practised

- Variables
- User input and output
- Modulus operator (%)
- Conditional statements (if-else)

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter an integer.
4. View the result.

## Sample Output

Enter an integer: 7

Odd number

## What I Learned

I learned how to use the modulus operator (%) to determine whether a number is odd or even and how to make decisions using an if-else statement.
