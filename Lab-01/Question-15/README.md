# Question 15

## Problem Statement

Write a C++ program that reads two floating-point numbers from the user and determines the larger number using the conditional (ternary) operator.

## C++ Concepts Practised

- Floating-point variables
- User input and output
- Relational operators
- Conditional (ternary) operator

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter two floating-point numbers.
4. View the larger number displayed.

## Sample Output

Enter first floating-point number: 12.5

Enter second floating-point number: 8.7

The larger number is: 12.5

## What I Learned

I learned how to compare floating-point values and use the conditional (ternary) operator to make simple decisions in a concise way.
