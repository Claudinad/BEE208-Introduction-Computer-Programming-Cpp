# Question 1

## Problem Statement

Write a C++ program that accepts two integer values from the user and calculates their sum, difference, product, and quotient using the appropriate arithmetic operators.

## C++ Concepts Practised

* Variables and data types
* Input and output using `cin` and `cout`
* Arithmetic operators (+, -, *, /)
* Conditional statements (`if` and `else`)
* Type casting using `static_cast<double>()`

## How to Run the Code

1. Compile the program using a C++ compiler.
2. Run the executable file.
3. Enter two integer values when prompted.
4. View the calculated results.

## Sample Output

Enter the first integer: 20

Enter the second integer: 5

Sum = 25

Difference = 15

Product = 100

Quotient = 4

## What I Learned

I learned how to use arithmetic operators in C++, accept user input, display output, and handle division by zero using an if-else statement.
