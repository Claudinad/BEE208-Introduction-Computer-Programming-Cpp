# Question 3

## Problem Statement

Write a C++ program that accepts three integer values from the user and checks whether they are equal using the equality operator.

## C++ Concepts Practised

* Variables
* User input and output
* Equality operator (==)
* Logical AND operator (&&)
* Conditional statements (if-else)

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter three integers.
4. View the result.

## Sample Output

Enter three integers: 5 5 5

Equal

## What I Learned

I learned how to compare multiple values using the equality operator and how to combine conditions using the logical AND operator.
