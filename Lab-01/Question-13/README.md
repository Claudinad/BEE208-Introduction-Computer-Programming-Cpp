# Question 13

## Problem Statement

Write a C++ program that swaps two integer values using the XOR operator and displays the values before and after swapping.

## C++ Concepts Practised

- Variables
- User input and output
- Bitwise XOR operator (^)
- Swapping values without a temporary variable

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter two integers.
4. Observe the values before and after swapping.

## Sample Output

Enter first number: 5

Enter second number: 8

Before swapping:
a = 5, b = 8

After swapping:
a = 8, b = 5

## What I Learned

I learned how the bitwise XOR operator can be used to swap two values without using an additional temporary variable.
