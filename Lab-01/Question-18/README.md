# Question 18

## Problem Statement

Write a C++ program that converts a binary number entered by the user into its decimal equivalent.

## C++ Concepts Practised

- Variables
- User input and output
- While loops
- Modulus operator (%)
- Mathematical functions using `pow()`
- Number system conversion

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a binary number.
4. View the decimal equivalent.

## Sample Output

Enter a binary number: 1010

Decimal equivalent = 10

## What I Learned

I learned how binary numbers are converted to decimal numbers and how loops and mathematical functions can be used to perform number system conversions.
