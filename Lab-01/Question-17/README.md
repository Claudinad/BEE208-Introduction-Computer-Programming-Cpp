# Question 17

## Problem Statement

Write a C++ program that determines whether a given number is a power of 2 using bitwise operators.

## C++ Concepts Practised

- Variables
- User input and output
- Bitwise AND operator (&)
- Conditional statements
- Binary number representation

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a positive integer.
4. View the result.

## Sample Output

Enter a number: 16

16 is a power of 2.

## What I Learned

I learned how bitwise operators can be used to solve problems efficiently and how powers of 2 can be identified using binary representation.
