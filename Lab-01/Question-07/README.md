# Question 7

## Problem Statement

Write a C++ program that reads a string from the user and determines whether it contains the character 'a'.

## C++ Concepts Practised

- Strings
- User input and output
- String searching
- Conditional statements
- `find()` function

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a string.
4. View the result.

## Sample Output

Enter a string: banana

Contains 'a'

## What I Learned

I learned how to search for characters within a string using the `find()` function and how to use conditional statements to display different outputs.
