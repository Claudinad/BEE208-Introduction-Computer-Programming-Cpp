# Question 4

## Problem Statement

Write a C++ program that reads a string from the user and checks whether it is equal to the string "hello". If it is equal, display "Hello!"; otherwise display "Goodbye!".

## C++ Concepts Practised

- String variables
- User input and output
- Equality operator (==)
- Conditional statements (if-else)

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a string.
4. View the result.

## Sample Output

Enter a string: hello

Hello!

## What I Learned

I learned how to compare strings using the equality operator and make decisions using an if-else statement.
