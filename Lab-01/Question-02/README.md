# Question 2

## Problem Statement

Write a C++ program that accepts the radius of a circle and calculates its area using the formula:

Area = πr²

## C++ Concepts Practised

* Variables and constants
* User input and output
* Mathematical calculations
* The `pow()` function from the `<cmath>` library

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter the radius of the circle.
4. View the calculated area.

## Sample Output

Enter the radius of the circle: 5

The area of the circle is: 78.5397

## What I Learned

I learned how to use mathematical formulas in C++, use constants, and apply the `pow()` function to calculate powers.
