# Question 12

## Problem Statement

Write a C++ program that reads a number from the user and determines whether it is both odd and divisible by 3.

## C++ Concepts Practised

- Variables
- User input and output
- Modulus operator (%)
- Logical AND operator (&&)
- Conditional statements (if-else)

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a number.
4. View the result.

## Sample Output

Enter a number: 9

The number is both odd and divisible by 3.

## What I Learned

I learned how to combine multiple conditions using the logical AND operator and use the modulus operator to test divisibility.
