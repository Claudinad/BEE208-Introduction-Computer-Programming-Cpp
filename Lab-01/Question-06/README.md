# Question 6

## Problem Statement

Write a C++ program that reads a character from the user and determines whether it is an uppercase letter.

## C++ Concepts Practised

* Character variables
* User input and output
* Logical operators
* Character checking using `isupper()`
* Conditional statements

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a character.
4. View the result.

## Sample Output

Enter a character: A

Uppercase letter

## What I Learned

I learned how to test characters in C++ and determine whether a character is uppercase using the `isupper()` function.
