# Question 10

## Problem Statement

Write a C++ program that reads two integers from the user and determines the larger number using the conditional (ternary) operator.

## C++ Concepts Practised

- Variables
- User input and output
- Relational operators
- Conditional (ternary) operator

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter two integers.
4. View the larger number displayed.

## Sample Output

Enter two integers: 12 20

The larger number is: 20

## What I Learned

I learned how to use the conditional (ternary) operator as a shorter alternative to an if-else statement when making simple decisions.
