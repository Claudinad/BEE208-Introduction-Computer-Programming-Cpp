# Question 19

## Problem Statement

Write a C++ program that accepts a binary number from the user, converts it to decimal, and then displays its hexadecimal and octal equivalents.

## C++ Concepts Practised

- Strings
- User input and output
- For loops
- Number system conversion
- Stream manipulators (`hex`, `oct`, `uppercase`)

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter a binary number.
4. View the decimal, hexadecimal, and octal equivalents.

## Sample Output

Enter a binary number: 1010

Decimal: 10

Hexadecimal: A

Octal: 12

## What I Learned

I learned how to convert binary numbers to decimal values and display numbers in hexadecimal and octal formats using C++ stream manipulators.
