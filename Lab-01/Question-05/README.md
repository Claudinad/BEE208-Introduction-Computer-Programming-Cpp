# Question 5

## Problem Statement

Write a C++ program that reads two integer values from the user and determines whether the first value is greater than the second value.

## C++ Concepts Practised

- Variables
- User input and output
- Greater-than operator (>)
- Conditional statements (if-else)

## How to Run the Code

1. Compile the program.
2. Run the executable file.
3. Enter two integers.
4. View the result.

## Sample Output

Enter two integers: 15 8

First value is greater

## What I Learned

I learned how to compare two values using the greater-than operator and use an if-else statement to make decisions.
